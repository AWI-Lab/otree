Zu Beginn jeder Runde sehen Sie eine Lotterie, die Ihnen zunächst zugeteilt wurde. Dies ist Ihre Anfangslotterie. Wenn Sie diese Anfangslotterie behalten möchten, müssen Sie nichts tun und lediglich {{ interval }} Sekunden warten. Die Runde endet dann und Ihre Anfangslotterie wird gespielt.

{% if default == "Safe" %}

Alternativ können Sie die Ergebnisse der Anfangslotterie ändern, indem Sie auf den Button „Anpassen“ klicken. Dazu haben Sie {{ interval }} Sekunden Zeit. Sobald Sie auf den Button „Anpassen“ geklickt haben, sinkt die Auszahlung, die Sie erhalten, wenn das Ergebnis „grün“ ausgewählt wird. Gleichzeitig steigt die Auszahlung, die Sie erhalten, wenn das Ergebnis „gelb“ angezeigt wird. Unter dem grünen Balken wird Ihnen angezeigt, um welchen Betrag das grüne Ergebnis sinkt sofern Sie auf den Button „Anpassen“ klicken. Unter dem gelben Balken wird Ihnen angezeigt, um welchen Betrag das gelbe Ergebnis steigt, wenn Sie auf den Button „Anpassen“ klicken.

Weiter unten auf dem Bild sehen Sie ein Beispiel. Im linken Bild wird Ihnen die ursprüngliche Lotterie angezeigt. Im rechten Bild sehen Sie diejenige Lotterie, die durch Klicken des Buttons „Anpassen“ entstehen würde. Klicken Sie in diesem Beispiel auf „Anpassen“, sinkt die Auszahlung für das grüne Ergebnis von 8 auf 7 Euro. Gleichzeitig steigt durch Klicken des Buttons „Anpassen“ die Auszahlung für das gelbe Ergebnis von 13 auf 14.50 Euro.

Wenn Sie auf „Anpassen“ geklickt haben, wird Ihnen die neue Lotterie nach Ablauf der {{ interval }} Sekunden angezeigt. Danach haben Sie wieder die Möglichkeit zu entscheiden, ob Sie diese neue Lotterie behalten wollen oder ob Sie einen weiteren Anpassungsschritt möchten. Wollen Sie die neue Lotterie behalten, müssen Sie wiederum {{ interval }} Sekunden warten. Die Lotterie wird dann automatisch ausgewählt und eines der Ereignisse (grün oder gelb) wird zufällig ausgewählt. Sofern Sie die neue Lotterie erneut anpassen wollen, klicken Sie bitte erneut auf „Anpassen“. Sie haben dazu wieder {{ interval }} Sekunden Zeit. Sie könne die Lotterie so lange anpassen, bis der grüne Balken (Ergebnis) 0 Euro erreicht. In diesem Fall wählen Sie diese Lotterie.
